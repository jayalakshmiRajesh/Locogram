Use Locogram's APIs to get structured product information from website of any women's clothing retailer in the US. We use machine learning techniques to automatically extract all categories, products and structured product information from a retailer's website.

2. How to Use:

Description
Make a call to the below url to get the list of products available for sale on retailer's website.

API Call

GET https://us.locogram.com/api/getProducts.json?name=XXX&url=YYY&email=ZZZ&token=ABC
Parameters 
XXX: Store Name 
YYY: Website url
ZZZ: developer email
ABC: developer token
Sample JSON Object 
'products': [
        {
            'id': '',
            'category': '',
            'subcategory': '',
            'title': '',
            'brand': '',
            'description':'',
            'url': '',
            'thumbnail': '',
            'image': '',
            'price': '',
            'sale_price': '',
            'variants': [{
                'color': '',
                'sizes': [],
                'images': [{
                    'thumbnail': '',
                    'image': ''
                }]
            }]
        }
